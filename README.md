## Descripción del Proyecto

El **Conversor de Moneda** es una aplicación de consola desarrollada en Java que permite convertir entre diferentes monedas como: Dólar, Peso Argentino, Real Brasileño y Peso Colombiano.  
Utiliza la **API de ExchangeRate** para obtener tasas de cambio actualizadas y la biblioteca **Gson** para procesar respuestas en JSON.

El proyecto incluye:  

- Menú interactivo por consola.  
- Conversión de cantidades ingresadas por el usuario.  
- Formato latinoamericano de números (puntos para miles, coma para decimales).  
- Manejo de errores y validación de entradas.
